ResponseMessage

In package com.globalkit.microapps.bus.core.domain

class com.globalkit.microapps.bus.core.domain.ResponseMessage

ResponseMessage is a data class which is instantiated any time whe need to generate a response for a request. This class indicates the id, the result body, any error detail (if any) and if the request treatment succeeded or not

Also look at RequestMessage, Bus

Fields

isSuccess: Boolean

indicates whether the request was successfully processed or not

id: String?

optional id for a response instance. This attribute is intended to contain the same id value at the request instance this response is instance is for. Te main goal is to establish a request-response relationship. Subscribers are the ones which create this response instances.

It is highly recommended that this attribute contains the value of the request id that generated this response

result: String?

optional Json formatted message which indicates the response for the processed request

var error: Throwable?

optional if there was an error, you could get the exception detail from this attribute
